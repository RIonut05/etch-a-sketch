# 🎨 Etch-A-Sketch

A browser-based drawing toy built with vanilla HTML, CSS, and JavaScript — no libraries, no frameworks, just pixels and vibes.

---

## Features
- Draw on a grid - hover your mouse over squares to colour them in
- Adjustable grid size choose how many squares per row/column (e.g. 16×16 up to 64×64)
- Clear / Reset - wipe the board and start fresh
- Classic mode - squares fill in solid black (the default)
- Rainbow mode - each square interaction applies a random RGB colour
- Darkening mode - each interaction darkens a square by 10%, building up gradually to full black over ~10 passes
